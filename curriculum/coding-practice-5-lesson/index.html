FlexBox Practice

You can do a lot of things using Flexbox, but mainly it is a way for you to lay content out on a page in an organized way. Watch this video for a very quick explanation of Flexbox (it's ok if not everything makes sense).

For example, by default, block level elements like div automatically stack veritcally:

Ex. 1:

Yellow Div
Red Div
Green Div

However, what if you wanted to have them lay out horizontally instead?

Ex. 2:

Yellow Div
Red Div
Green Div

Here's How:

  1. First, ensure that you have created a new folder in your VS Code project called coding-practice-5 Ensure the folder has index.html and styles.css
  2. In index.html, Create a div element and make sure to open it up.
  3. Give the div element a class attribute of colored-boxes-container
  4. Inside the div, create 3 more child div elements, and make sure to open them up. Then:
    • Write the text "Yellow Div" inside the first child div element. Then, give it a class attribute of "yellow"
    • Write the text "Red Div" inside the second child div element. Then, give it a class attribute of "red"
    • Write the text "Green "Div" inside the third child div element. Then, give it a class attribute of "green"
  5. In styles.css, write CSS code that does the following:
    • Turn the background-color of the "Yellow Div" yellow
    • Turn the background-color of the "Red Div" red
    • Turn the background-color of the "Green Div" green
  6. In styles.css, write code that gives .colored-boxes-container a display property of flex

If you did this correctly, your output should be side-by-side like Ex. 2 above. It's ok if it doesn't look exactly the same.
