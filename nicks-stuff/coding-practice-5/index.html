FlexBox Practice

You can do a lot of things using Flexbox, but mainly it is a way for you as a coder to lay content out on a page in an organized way.

For example, by default, block level elements like div automatically stack veritcally:

Yellow Box
Red box
Green Box

However, what if you wanted to have them lay out horizontally instead?

Yellow Box
Red Box
Green Box

Here's How:

  1. In index.html, Create a div element and make sure to open it up.
  2. Give the div element a class attribute of colored-boxes-container.
  3. Inside the div, create 3 more child div elements, and make sure to open them up.
    • Give the first child div element a class attribute of "yellow"
    • Give the second child div element a class attribute of "red"
    • Give the third child div element a class attribute of "green"
  4. Place a p tag inside each of the child div elements you just created. Then:
    • Put the text "Yellow Box" in the first p tag
    • Put the text "Red Box" in the second p tag
    • Put the text "Green Box" in the third p tag
